Config

main.js

default_dir_prefix
Define hidden directories prefix. (Will be ignored by Router)
Default: @
path.view
View include path. Must be relative from app/ directory.
Default: undefined
path.template
Template include path. Must be relative from app/ directory.
Default: undefined

routing.js

Overrides URL call to other path. 


"regexp_string/(.*)": {
  path: "path/to/view",
  param: {
    "sample": "Value"
  }
}

//=> new RegExp('^('+ regexp_string +')$');
